# 바른 컨설팅 - 한의원 경영 분석 플랫폼 (wealthywei)

이 프로젝트는 한의원 매출 데이터를 Excel에서 정밀하게 추출하고 분석하여, 성과 지표와 컨설팅 리포트를 제공하는 서비스입니다.

## 🚀 주요 기능
- **정밀 엑셀 데이터 엔진**: 7행 규칙 및 지능형 키워드 매핑을 통한 정확한 데이터 추출.
- **Toss 스타일 대시보드**: 현대적이고 깔끔한 UI로 매출 현황(총매출, 보험매출, 비급여 등) 한눈에 파악.
- **실시간 비교 분석**: 상세 페이지(/details)에서 기준월과 대비월을 즉시 선택하고 비교하는 성과 리포트.
- **애니메이션 피드백**: 숫자 카운팅 효과(Number Counter Effect) 및 반응형 그래프.

## 🛠️ 기술 스택
- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **Chart**: Recharts
- **Icons**: Lucide React
- **Data Parsing**: XLSX

## 🏁 시작하기

먼저, 개발 서버를 실행합니다:

```bash
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 결과를 확인하세요.
